This project is used to test out backbone model data assignment.

# Commands

1. php server

    ```bash
    php -S localhost:9001 -t ./php-backend/public/
    ```

2. backbone frontend

    ```bash
    php -S localhost:9002 -t ./backbone-frontend/
    ```
